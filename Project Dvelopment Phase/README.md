# Electric Motor Temperature Prediction using Machine Learning

This project predicts electric motor rotor temperature using Machine Learning algorithms.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask

## Models Implemented
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Support Vector Regression

Decision Tree model achieved the best accuracy and was deployed using Flask.

## How to Run
1. Install requirements
2. Run:
   python app.py
3. Open browser:
   http://127.0.0.1:5002

## Output
Predicts Permanent Magnet Surface Temperature of Electric Motor.
